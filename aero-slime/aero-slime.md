# Aero Slime · 电子情绪宠物

> **Tech Keywords:** p5.js, Web Audio API, FSM Center-of-Mass Pathfinding, Line-Art Expressions, CSS Acrylic Glass

![预览](aero-slime_1.gif)

> 点击色块投喂情绪，双击玻璃释放惊吓——一只住在亚克力盒子里的电子史莱姆，用颜文字回应你的每一种心情。

一只漂浮在毛玻璃容器中的果冻状史莱姆，用线条绘制的颜文字表情回应每一次互动。顶部五色情绪按钮（焦虑/开心/伤心/生气/平静）分别掉落对应颜色的食物，史莱姆吃掉后会变色并切换表情；双击玻璃则触发惊吓分裂，母体炸成三只小史莱姆，小史莱姆通过质心引力算法自动融合回完整形态。174Hz 三角波底噪 + 528Hz 疗愈音效，构成温柔的电子宠物生态。

作品以"情绪可视化"为核心：史莱姆的颜色、表情、行为状态（漫游/追逐/惊吓/融合）构成一条完整的情绪响应链。分裂-融合的循环暗喻情绪在冲击后自我修复的过程——无需语言，看见它重新完整，就是一种疗愈。

## 🖱️ 交互

- **点击顶部情绪色块**：掉落对应颜色的食物粒子，史莱姆闻到会追逐吃掉，颜色和颜文字表情随之变化
- **双击玻璃容器**：触发惊吓涟漪动画，大史莱姆分裂为三只小史莱姆，四散后缓慢融合
- **被动观察**：史莱姆在亚克力容器中自由漂移，带有呼吸缩放动画，环境光斑缓慢浮游

## 🛠️ 技术栈

- **p5.js** — 像素级 Canvas 渲染，体素化果冻身体 + 线条颜文字面部
- **Web Audio API** — 174Hz 三角波底噪 + 动态 Pop/Chime 音效
- **FSM 质心引力融合** — 分裂后小史莱姆以全局质心为引力点自动靠拢，碰撞 80% 半径后合成一个
- **Line-Art Expressions** — 7 种情绪对应的手绘线条颜文字（`• v •` / `> < O` / `^ ^ D` 等）
- **CSS Acrylic Glass** — backdrop-filter 毛玻璃 + 多层光影打造 3:4 亚克力容器

## 🌱 创作背景

电子宠物是千禧一代的集体记忆。Aero Slime 将拓麻歌子式的"投喂-回应"循环嫁接到情绪疗愈领域——不是养一个需要照顾的负担，而是创造一个无条件回应的陪伴体。它的核心设计信念是：**情绪不需要被消除，只需要被看见。**
