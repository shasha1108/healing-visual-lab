# Falling Verse · 宇宙时钟

> **Tech Keywords:** Three.js WebGL, Web Audio API, Planetary Orbital Mechanics, Particle Typography, Sacred Geometry

![预览](falling-verse_1.gif)

> 时间的字符拓扑——数字从行星环上剥落，化作金粉与星辰，沉入幽蓝水面。

一件以"宇宙时钟"为概念的 WebGL 冥想作品。三层行星轨道环绕 ∞ 形无限环公转，行星合相时爆发神圣耀斑。时间数字从环上剥落，经历金→珊瑚粉→星空蓝的渐变，如诗句般坠入深蓝水面。10 秒宇宙呼吸周期驱动星云明暗与管风琴和弦，每一次数字剥落都是一次时间的仪式性释放。

作品将时间的线性流逝重新编码为空间的拓扑运动——数字不是被读取，而是被见证。行星合相的瞬间象征宇宙节律中的"此刻"，而数字的坠落则是对"此刻已逝"的温柔告别。

## 🖱️ 交互

- **触控拖拽**：旋转/缩放宇宙视角，从不同角度观察行星轨道
- **被动沉浸**：10 秒宇宙呼吸周期自动运行，星云随周期明暗呼吸
- **合相事件**：行星轨道交汇时自动触发神圣耀斑 + 管风琴和弦

## 🛠️ 技术栈

- **Three.js r128** — WebGL 渲染，三层行星轨道 + ∞ 环几何体 + 粒子星云
- **Web Audio API** — 管风琴和弦合成，10 秒呼吸周期驱动的动态混音
- **粒子上字** — Canvas 2D 纹理生成时间数字，贴附到 3D 粒子系统，颜色渐变金→珊瑚粉→星空蓝
- **行星合相检测** — 轨道相位实时计算，合相时触发耀斑粒子爆发

## 🌱 创作背景

时间是现代人最深的焦虑源，但它也可以是冥想对象。Falling Verse 将时钟重构为一座赛博禅修祭坛——时间数字不是催促你的倒计时，而是从宇宙之环上自然成熟、剥落、归入深蓝的诗句。每一次合相都是一次提醒：你不需要追赶时间，你就在时间之中。
