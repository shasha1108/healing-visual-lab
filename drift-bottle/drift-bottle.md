# drift-bottle · 平行世界 · 漂流瓶生态系统

> **Tech Keywords:** p5.js, Canvas 2D Wave Physics, Clip Pipeline, BlendMode Screen, Web Audio API, Frutiger Aero

![预览](drift-bottle_1.gif)

> 轻触漂流瓶，唤醒瓶中平行世界——帆船、发光鲨鱼、微型海鸥在同一片玻璃海域中共生。

一件以"瓶中平行世界"为核心的 H5 互动作品。画面中央一只漂浮的药剂瓶倾斜摇曳于风浪之上，瓶内自成一个独立的微缩生态系统：帆船往复巡航、深海鲨鱼灯笼巡游、海鸥翱翔、浮游荧光粒子缓缓升腾。外海采用 Trochoidal 波浪物理模拟和五层高精波层叠加，阳光耀斑与海面波光交相辉映。

点击画面即可敲响漂流瓶：瓶身随之晃动，涟漪以触点为中心扩散，瓶内生态系统瞬间加速——鲨鱼冲刺、浮游粒子爆发、空灵和弦与水泡音效同时触发。每一次触碰都是一次对瓶中世界的叩问。

## 🖱️ 交互

- **点击 / 触摸画面**：敲响漂流瓶，触发涟漪扩散 + 空灵和弦 + 水泡音效
- **瓶身晃动**：每次点击瓶身产生物理级倾斜回弹，瓶内水面保持重力解耦
- **生态系统加速**：鲨鱼冲刺 1.2 秒、浮游粒子爆发 6 颗、帆船继续巡航
- **自动循环**：无操作时，外海浪涌、海鸥翱翔、云朵飘移持续运行

## 🛠️ 技术栈

- **p5.js** — Canvas 2D 渲染引擎
- **Canvas 2D Clip Pipeline** — 瓶形遮罩内独立渲染平行生态
- **BlendMode Screen** — 玻璃高光与鲨鱼灯笼发光层
- **Web Audio API** — 程序化生成空灵和弦 + 水泡涌动音效
- **CSS Keyframes** — 环境光球体漂浮动画 + 涟漪消散动画
- **Trochoidal Wave Physics** — 五层高精海浪模拟 + 波光闪烁粒子

## 🌱 创作背景

漂流瓶是跨越时空的匿名对话。这件作品将漂流瓶重新想象为一个"平行世界"的容器——瓶内并非一纸信笺，而是一个完整的、自洽的微型生态。帆船、鲨鱼、海鸥、浮游生物在玻璃弧面内共生，外海与内海的波浪各自独立又彼此呼应。每一次点击，都是两个世界之间的一次共振。
